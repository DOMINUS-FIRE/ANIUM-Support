ANIUM SUPPORT BOT — ВЕРСИЯ ДЛЯ RENDER
=====================================

Что исправлено
--------------
1. Бот открывает HTTP-порт 0.0.0.0:$PORT для Render Web Service.
2. Добавлены страницы / и /health.
3. Исправлены ответы от имени канала или группы через sender_chat.
4. Если локальная SQLite-база очистилась после перезапуска Render,
   бот может определить пользователя по #id прямо в обращении.
5. В группу теперь отправляется ID пользователя внутри текста или подписи.
6. При ошибке ответа бот пишет понятное сообщение в группу.
7. Обновлено приветствие /start.

КАК ЗАГРУЗИТЬ НА RENDER
-----------------------

1. Загрузите в корень GitHub-репозитория файлы:
   - bot.py
   - requirements.txt
   - render.yaml
   - Procfile
   - .gitignore

2. На Render выберите:
   New -> Web Service

3. Подключите GitHub-репозиторий.

4. Настройки:
   Language: Python 3
   Build Command: pip install -r requirements.txt
   Start Command: python bot.py

5. В Environment добавьте:
   BOT_TOKEN = токен от @BotFather
   SUPPORT_CHAT_ID = @thfjffhf

   PORT вручную добавлять не надо — Render создаёт его автоматически.

6. В Health Check Path укажите:
   /health

7. Нажмите Deploy Web Service.

КАК ОТВЕЧАТЬ
-------------
В группе ответьте свайпом вправо именно на обращение пользователя,
где внизу указан #id123456789.

Можно отвечать:
- от личного аккаунта администратора;
- от имени канала;
- от имени группы.

После успешной доставки бот поставит реакцию ✅.
Если реакции в группе запрещены, бот напишет:
"✅ Ответ отправлен пользователю."

ВАЖНО
-----
- Бот должен быть добавлен в группу @thfjffhf.
- Лучше назначить его администратором.
- Пользователь должен сначала открыть личный чат с ботом.
- Не запускайте одновременно две копии бота с одним токеном.
- При использовании тем заполните SUPPORT_THREAD_ID.
